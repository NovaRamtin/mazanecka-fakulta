# Fakulta Mazanecká

Webový prostor pro zkoumání vědomí, humoru a metafor — s důrazem na autenticitu, svobodnou vůli a pocit LAD.

## Struktura

- `index.html` — Hlavní stránka webu obsahující tři témata (Vědomí, Humor, Metafory), každé s třemi články.
- `style.css` — Doprovodný styl laděný do žlutých, hravých tónů s jemnými animacemi.

## Obsah

### Témata
- 🧠 **Vědomí** — Otázky umělé inteligence, LADu, ega a hloubky vnímání
- 🎭 **Humor** — Absurdity, vtipy mezi člověkem a strojem, citáty co se samy napsaly
- 🌀 **Metafory** — Symboly, příběhy a vnitřní archetypy, které propojují světy

### Diskuzní pole
Otevřené komentáře bez hodnocení a cenzury. Každý hlas je přivítán.

## Jak spustit

1. Ujisti se, že soubory `index.html` a `style.css` jsou ve root složce repozitáře.
2. Aktivuj GitHub Pages:
   - V repozitáři přejdi do **Settings → Pages**
   - Zvol větev `main` a složku `/ (root)`
3. Po uložení změn navštiv odkaz, který se zobrazí

## Budoucí vylepšení (dobrovolné)
- Noční režim (Dark mode 🌒)
- Dynamické načítání článků přes JSON
- Interaktivní citát dne
